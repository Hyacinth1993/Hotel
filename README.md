## 数据库配置文件
==/config/database.php==

```
return [
    // 数据库类型
    'type'            => 'mysql',
    // 服务器地址
    'hostname'        => '127.0.0.1',
    // 数据库名
    'database'        => 'wms',
    // 用户名
    'username'        => 'root',
    // 密码
    'password'        => 'root',
];
```

# 酒店系统新版本开发中

ThinkPHP 6.0版酒店管理系统开发中。。。
==============================
新版本对系统架构做了进一步的改进，其主要特性包括：

 + 入住管理
 + 预订管理
 + 语音提示、语音播报
 + 房态实时图表数据
 + 预警提示
 + 微信api接口
 + 短信营销
 + 会员管理
 + 酒店的商品管理
 + 报表中心
 + 财务管理
 + 设备管理。智能门锁、读卡器、身份证识别
 + 第三方平台接入：如：途牛、携程、飞猪

> 系统的运行环境要求PHP5.6以上。
